## What Is Tagumi?

Tagumi is a local-first desktop tag editor for music collections. It is built for fast single-file edits, safe batch workflows, format-correct metadata handling, and long cleanup sessions across large libraries.

## Features

- Edit common metadata such as title, artist, album, album artist, track number, disc number, year, genre, BPM, composer, publisher, comment, and lyrics.
- Edit extended metadata including subtitle, grouping, conductor, remixer, original artist, original album, ISRC, mood, key, language, and copyright.
- Work on one file or many files at once with merged multi-selection editing.
- Add, replace, remove, copy, paste, resize, and validate album art before saving.
- Import cover art from local files, a URL, or iTunes search inside the app.
- Detect BPM from audio or tap tempo manually.
- Rename files from tag-based templates with previews and saved presets.
- Reorder tracks and write sequential track numbers with optional totals.
- Run bulk find-and-replace and capitalization workflows with previews.
- Build multi-step automations with conditions, cover art updates, tag edits, and file renames.
- Preview audio, surface read errors, and undo the most recent save, rename, track-number, or automation operation.
- Use the app in English, Japanese, Spanish, Portuguese, Simplified Chinese, and Traditional Chinese.

## Supported Formats

| Format | Tag Standard |
| --- | --- |
| MP3 | ID3v1 / ID3v2 |
| FLAC | Vorbis Comments |
| OGG | Vorbis Comments |
| M4A / AAC | iTunes / MP4 atoms |
| OPUS | Vorbis Comments |
| WAV | ID3v2 / RIFF INFO |
| AIFF | ID3v2 |
| WavPack | APEv2 |
| Musepack | APEv2 |
